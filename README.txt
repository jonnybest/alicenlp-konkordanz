KWIC creation tool
==================
A tool for creating keyword-in-context (KWIC) overviews for a given corpus.

This program is part of the AliceNLP project http://svn.ipd.kit.edu/trac/AliceNLP/

Maven:

<dependency>
  <groupId>edu.kit.ipd.aliceNlp</groupId>
  <artifactId>konkordanz</artifactId>
  <version>0.5</version>
</dependency>
